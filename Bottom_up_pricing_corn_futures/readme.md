The strategy begins from that precise principle:

  If a model can estimate the physical state of the corn market more consistently than the market has priced it,
  then futures and calendar spreads can be traded when market prices move far enough away from model implied fair value.

The trading problem can be separated into three connected layers. The first layer estimates the physical state of the corn market,
including supply, demand, inventories, stocks to use, positioning, seasonality, and risk conditions.
The second layer translates that state into a fair value for each futures contract, 
because a July contract and a December contract do not represent the same economic exposure.
July is more closely linked to old crop inventory and nearby scarcity, while December is more closely linked to new crop acreage, yield, and harvest expectations.
The third layer trades only when the gap between market price and fitted fair value is large enough to compensate for model error, execution cost, and risk.
